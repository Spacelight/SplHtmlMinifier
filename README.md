# SplHtmlMinifier
Tiny HTML Minifier - .NET / C# / .ASP MVC

0.2 Version
Tested mostly visually & some cleanups

### License: MIT

### Pros:
* Almost no regexes - parsing is fast
* At most 2 parsed HTML tags kept live in heap - tiny memory footprint

### Cons:
* Doesn't touch JS or CSS
* No settings
* Needs more testing
